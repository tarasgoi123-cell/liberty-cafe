Наші переваги

Чому обирають Liberty

🫘

Спеціальна кава

Працюємо лише з перевіреними ростерами. Кожен лот — з конкретної ферми, з відомою датою збору.

👨‍🍳

Досвідчені бариста

Наші бариста — учасники всеукраїнських чемпіонатів. Кожна чашка — результат сотень годин практики.

🏡

Живий простір

Місця для роботи з ноутбуком, затишні куточки для зустрічей, швидкий Wi-Fi і розетки скрізь.

🌱

Відповідально

Рослинне молоко без доплати, біорозкладна упаковка, підтримка локальних виробників.
